#### 자연어 유사도 측정을 어떻게 하셨나요??
문서를 embedding하는 것과 같은 원리로 문서를 parsing해서 텍스트를 추출하고,
거기서 chunking해서 작은 단위로 분할하고 embedding 생성해서 벡터로 변환한 후,
캐시 유사도 점수가 스레스홀드 이상되었을 때만 캐시가 응답하도록 설정되어 있습니다.
일단 스레스홀드는 시연의 예시에 맞춘거고 다양한 곳에 활용하기 위해서는 나중에 develop해야할 것

#### AI Agent 가드레일도 적용이 되어있나요??
AI Agent 가드레일 노드 설정은 없고, 출력값의 형식이 잘못된 경우 등에 대해서만 막고 있음.
다만 이런 경우에 유저가 로그에서 어떤 부분이 잘못되었는지는 확인할 수 있도록 구성.
