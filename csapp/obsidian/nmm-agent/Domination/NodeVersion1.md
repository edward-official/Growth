## 노드 설명

| 노드 타입 (Type ID)       | 노드 이름                            | 설명                                         |
| :-------------------- | :------------------------------- | :----------------------------------------- |
| `start`               | 시작 노드 (Start Node)               | 워크플로우의 진입점이며, 사용자 입력을 받아 워크플로우를 시작합니다.     |
| `end`                 | 종료 노드 (End Node)                 | 워크플로우의 실행을 종료하고 최종 결과를 반환합니다.              |
| `llm`                 | LLM 노드 (LLM Node)                | 대형 언어 모델(LLM)을 호출하여 텍스트 응답을 생성합니다.         |
| `knowledge-retrieval` | 지식 검색 노드 (Knowledge Retrieval)   | 벡터 데이터베이스(RAG)에서 사용자 질문과 관련된 문서를 검색합니다.    |
| `if-else`             | 조건 분기 노드 (If-Else Node)          | 설정된 조건에 따라 워크플로우의 실행 경로를 분기합니다.            |
| `question-classifier` | 질문 분류 노드 (Question Classifier)   | 사용자 질문의 의도를 분석하여 적절한 처리 경로로 분류합니다.         |
| `tavily-search`       | Tavily 검색 노드 (Tavily Search)     | Tavily API를 사용하여 웹 검색을 수행하고 실시간 정보를 수집합니다. |
| `assigner`            | 변수 할당 노드 (Assigner Node)         | 워크플로우 내 변수 값을 설정하거나 데이터를 변환합니다.            |
| `answer`              | 답변 노드 (Answer Node)              | 사용자에게 최종 답변을 생성하고 스트리밍 등을 처리합니다.           |
| `slack`               | 슬랙 노드 (Slack Node)               | Slack 채널로 메시지를 전송하거나 알림을 보냅니다.             |
| `http`                | HTTP 요청 노드 (HTTP Node)           | 외부 API 서버에 HTTP 요청을 보내고 응답을 처리합니다.         |
| `template-transform`  | 템플릿 변환 노드 (Template Transform)   | 템플릿 엔진을 사용하여 데이터를 특정 형식의 텍스트로 변환합니다.       |
| `imported-workflow`   | 가져온 워크플로우 노드 (Imported Workflow) | 다른 워크플로우를 서브루틴처럼 호출하여 실행합니다.               |


## 답변 노드

**답변 노드(`answer`)** 는 워크플로우의 중간 또는 최종 단계에서 사용자에게 보여줄 응답 메시지를 생성하는 노드입니다.

### 역할
LLM의 생성 결과, 검색된 데이터, 또는 다른 노드의 출력을 조합하여 사용자에게 전달할 최종 텍스트를 생성합니다.

### 주요 기능
- **템플릿 렌더링**: `{{ node_id.output }}` 문법을 사용하여 다른 노드의 출력 값을 텍스트 템플릿에 동적으로 삽입할 수 있습니다.
- **스트리밍 지원**: 생성된 답변을 실시간 스트리밍 형태로 사용자에게 전달할 수 있습니다.
- **유연한 연결**: 실행 경로상에 존재하는 어떤 노드의 출력값이라도 참조하여 답변을 구성할 수 있습니다.

### 입력 / 출력
**입력 포트**: `target`: 실행 순서를 제어하기 위한 포트 (선택 사항)
**출력 포트**: `final_output`: 템플릿 렌더링이 완료된 최종 응답 문자열
