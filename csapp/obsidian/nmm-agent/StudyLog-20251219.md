## LangChain: LLM(OpenAI, Claude 등)을 중심으로 “체인(Chain)” 형태의 로직을 쉽게 구성하게 해주는 라이브러리
#### 매번 반복되는 작업 표준화:
- 프롬프트 관리
- 여러 단계의 질문/응답 연결
- 외부 데이터 연동 (DB, PDF, WEB)
- 메모리(대화맥락) 유지
- 툴 호출 (API, 함수 실행)

#### 핵심 개념:
- `LLM`: OpenAI, Anthropic, Gemini 등 모델 추상화
- `Prompt Template`: 프롬프트를 변수화해서 재사용
- `Chain`: 입력 → 처리 → 출력이 고정된 파이프라인
- `Memory`: LLM은 기억력이 없기 때문에 우리가 대신 대화의 컨텍스트를 기억하고 프롬프트에 붙여줘야한다.
- `Tool/Agent`: Tool이라는 건 LLM이 할 수 없는 코드 실행, 검색, API 호출을 대신하는 외부 기능이고 Agent라는 건 이런 기능을 사용하는 타이밍을 스스로 판단하는 LLM을 말합니다.
