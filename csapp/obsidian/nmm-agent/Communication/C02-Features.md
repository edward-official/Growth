# 2. 핵심 기능

  

우리가 만드는 플랫폼은 복잡한 코딩을 몰라도, 누구나 강력한 AI 프로그램을 만들 수 있도록 다음과 같은 기능들을 제공합니다.



## 2.1. 눈으로 보며 만드는 작업 흐름 (Visual Workflow)

가장 핵심적인 기능으로, 마치 레고 블록을 조립하듯 화면에서 드래그 앤 드롭으로 AI의 동작 순서를 그릴 수 있습니다.

* **다양한 블록(노드) 지원**:

* **AI 대화 블록 (LLM Node)**: AI에게 질문하고 답변을 받는 단계.

* **지식 검색 블록 (Knowledge Retrieval Node)**: 미리 학습된 문서에서 정보를 찾아오는 단계.

* **코드 실행 블록 (Code Node)**: 복잡한 계산이나 데이터 처리가 필요할 때 사용하는 프로그래밍 단계.

* **조건 분기 블록 (If-Else Node)**: "만약 ~라면 A, 아니면 B"처럼 상황에 따라 다른 행동을 하게 만드는 단계.

* **도구 사용 블록 (Tool Node)**: 구글 검색이나 위키백과 같은 외부 기능을 가져와 사용하는 단계.

* **실시간 테스트**: 각 블록이 제대로 작동하는지 바로바로 눌러서 확인해볼 수 있습니다.

  

## 2.2. 나만의 지식 데이터 연동 (RAG Pipeline)

사용자가 가진 문서나 데이터를 AI가 읽고 이해할 수 있도록 쉽게 넣어주는 기능입니다.

* **데이터 업로드**: PDF, 메모장 파일, 웹사이트 주소 등을 올리면 AI가 내용을 읽습니다.

* **알아서 정리하기**: 긴 문서는 AI가 읽기 좋게 자동으로 자르고(Chunking) 다듬어줍니다.

* **AI 기억 저장소**: 텍스트를 AI가 이해하는 형태(벡터)로 바꿔서 저장합니다.

* **똑똑한 검색**: 단순히 단어만 찾는 게 아니라, 문맥과 의미를 파악해서 정확한 정보를 찾아냅니다.

  
  

## 2.3. 다양한 AI 모델 골라 쓰기 (Model Management)

특정 회사 AI에 얽매이지 않고, 상황에 맞춰 좋은 모델을 골라 쓸 수 있습니다.

* **유명 모델 지원**: 챗GPT(GPT-4), 클로드(Claude), 제미나이(Gemini) 등 유명한 AI들을 입맛대로 골라 쓸 수 있습니다.

* **내 컴퓨터 모델 지원**: 비싼 서버가 아니라 내 컴퓨터나 사내 서버에 설치한 AI 모델(Ollama 등)도 연결할 수 있습니다.

  
  

## 2.4. 유연한 배포와 연결 (Deployment & Integrations)

여기서 만든 AI 기능을 다른 홈페이지나 앱으로 쉽게 가져갈 수 있습니다.

* **간편한 연결 (API)**: 다른 프로그램에서 바로 호출해서 쓸 수 있는 주소(API)를 자동으로 만들어줍니다.

* **로컬 설치 지원 (Docker)**: 보안이 중요한 데이터 센터 내 폐쇄적 환경에서도 안전하게 사용할 수 있도록 도커(Docker) 기반의 설치를 지원합니다.
